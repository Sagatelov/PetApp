## About
Pet App is a mobile application developed to test various approaches to iOS app development. It is designed to interact with the JSONPlaceholder test API and includes user authentication through Firebase. Users can browse, create, edit, and delete data provided through the API, such as posts, comments, users, and more.

## Functionality

- **Data Viewing:** Users can browse a list of data retrieved from the JSONPlaceholder API in various categories like posts, comments, users, etc.

- **Registration and Authentication:** Users can register and log into the system using Firebase Authentication.

   <img src="https://i.ibb.co/Dtxk6r4/Frame-12.png" width="726">

- **Liked Posts Screen:** Implemented a screen where users can view posts they have liked.

   <img src="https://i.ibb.co/xLrsdhD/Frame-10.png" width="726">
</p>

- **Data Creation:** Users can create new posts, comments, and other objects by sending requests to the API.
- **Data Editing:** Users can edit existing data, such as modifying the text of posts or comments.
- **Data Deletion:** Users can delete existing posts and comments through the application.
- **Local Data Storage:** Use UserDefaults for storing user settings and CoreData for local data storage.
- **MVVM:** Implementation of the MVVM pattern to separate the application's logic into model, view, and view-model.
- **Programmatic UI and XIB Files:** Creating screens using programmatic UI for flexible interface customization, as well as using XIB files for creating custom views.
- **SOLID:** Application of SOLID principles to create a flexible and extensible application architecture.
- **Custom Views:** Creation of custom views for reusability and improved modularity of the application.

## Technologies

- **Programming Language:** Swift.
- **Frameworks:** Firebase for user authentication, UserDefaults for storing settings, CoreData for local data storage, UIKit for the user interface.
- **Architecture:** MVVM.
- **Interface:** Programmatic UI, XIB files, and custom views for creating application screens.
